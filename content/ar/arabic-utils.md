## **Morphology**

### **Free software**

- [AraMorph - Perl](https://sourceforge.net/projects/aramorph/) - An Arabic morphological analyzer and part-of-speech tagger written in Perl (originally by Tim Buckwalter)
- [AraMorph - Java](http://www.nongnu.org/aramorph/) - An Arabic morphological analyzer and part-of-speech tagger rewritten in Java for[ Lucene](http://lucene.apache.org/)
- [AraComLex](http://sourceforge.net/projects/aracomlex/) - An open source finite state morphology for Modern Standard Arabic. The source files can be compiled by the open source compiler, foma, or Xerox xfst.
- [UralicNLP](https://github.com/mikahama/uralicNLP) is a Python library that provides morphological tagging, generation, lemmatization and disambiguation in many languages including Arabic

### **Proprietary**

- [Xerox Arabic Morphological Analyzer and Generator](http://www.arabic-morphology.com/)

## **Parsers**

### **Free software**

- [Bikel's implementation of Collins Parser](http://www.cis.upenn.edu/~dbikel/software.html#stat-parser) by[ Dan Bikel](http://www.cis.upenn.edu/~dbikel/).
- [Arabic dictionaries](http://www.ling.ohio-state.edu/~jonsafari/arabiclg/arabiclg.20060829.tar.bz2), by[ Jon Dehdari](http://www.ling.ohio-state.edu/~jonsafari/), for the[ Link-Grammar parser](http://www.abisource.com/projects/link-grammar/). These require the Aramorph stemming package, above.
- [ElixirFM](https://sourceforge.net/apps/trac/elixir-fm/wiki) ([online interface here](http://quest.ms.mff.cuni.cz/cgi-bin/elixir/index.fcgi)) is a Functional Arabic Morphology written in Haskell and Perl; the lexicon is a "re-processed" version of the Buckwalter analyser.
- [Sarf](http://sourceforge.net/projects/sarf) - Arabic Morphology System (all in Java)

## **Diacritization**

### **Free software**

- [Miskhal](https://www.tahadz.com/mishkal/)
- [hAraCat](https://github.com/mikahama/haracat) a free tool for predicting vowels and other diacritics.

### **Free/open licence**

- [Meedan-Memory](http://github.com/anastaw/Meedan-Memory), Arabic-English TMX (sentence-aligned), ~467,000 words on the English side,[ Open Database Licence](http://www.opendatacommons.org/licenses/odbl/)
- [Quranic Arabic Corpus](http://quran.uk.net/), 77,430 words of Quranic Arabic, with manually verified contextual POS, inflection, derivation;[ dependency grammar](https://ar.wikipedia.org/wiki/Dependency_grammar) annotation is planned.
- [Arabic NER corpora](http://www1.ccls.columbia.edu/~ybenajiba/downloads.html) by[ Yassine Benajiba](http://www1.ccls.columbia.edu/~ybenajiba/), 150,000+ words.
- [UN parallel corpora](http://www.euromatrixplus.net/multi-un/)
- [HamleDT](http://ufal.mff.cuni.cz/hamledt), harmonized dependency treebanks of many languages, common annotation style.
